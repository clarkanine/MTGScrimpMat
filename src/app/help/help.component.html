*Attention*

This web application has only been run and tested on Google Chrome. The following instructions assume that is the browser it is running on. That's not to say it won't work on other browsers, I just have yet to test and support them.

Setting up your players

Go to the Player List page from the menu on the left. If you haven't logged in to your own account, you won't see your current players, so start adding them! Pressing the New Player button will bring up a dialog where you can enter player information, including a deck list. The deck list must be formatted like the following
4 Snapcaster Mage
4 Serum visions
.
.
.
i.e. the quantity of the card followed by its name. MTGGoldfish and TappedOut have an easy way to get lists of existing decks in this format.

Using the interface

Once you've added your players, navigate to the Current Game page from the menu on the left. Currently, MTGScrimp supports two-player magic games. You can select a player from the drop down list of players that you just created. Selecting a player will auto populate the form fields with information, as well as the corresponding deck list tab in the content below.

Saving player data

MTGScrimp is currently only capable of saving files locally. When you press the Save Player button, it will start to download a file for each form field.
Note: You will want to be able to download multiple files at once and have them overwrite existing files. To do that, checkout the Overwrite Downloads Extension and Configure Downloads links above. It is helpful to download these files to a well named folder such as MTGScrimpTextFiles.

Setting up OBS

Go ahead and click on the Save Player buttons for both of the players. You should see downloads for text files named someting like player1life.txt. To get these showing on OBS, create a text source that reads from its corresponding file that was created. Move them around until it looks pretty. Maybe you have a neat overlay to make it real fancy. Try changing some life totals and pressing Save Player; it should save the values and show up on your OBS window.

Saving cards

If you have a reason to display a card image on your stream, you can do that pretty easily with MTGScrimp. If the active players have deck lists, the cards will show up in their corresponding tabs in the bottom area. You can right-click on the image and Save As (preferably to the folder that has your text files to keep things tidy). If your players don't have deck lists, you can simply search for a card using the Card Search tab. Set up an Image Source in OBS from that file you just downloaded. Under each card is a link to its oracle text in case you need clarification when a heated argument begins during a game.
